# 🌲 소나무재선충병 피해 예측 — Fisher-KPP + PINN  (2016–2023)

산림청 **병해충발생정보관리** 연도별 자료(2016–2023)를 이용해, 반응-확산 방정식
**Fisher–KPP** 를 **물리정보신경망(PINN)** 으로 풀어 감염목(피해)의 시·공간 확산을
학습하고 **미래를 예측**합니다. 모든 결과는 **대한민국 지도 위에 매핑**됩니다.

## 지배방정식

$$\frac{\partial u}{\partial t} = D\,\nabla^2 u + r\,u(1-u)$$

- `u(x,y,t)` : 정규화 감염밀도 (0=청정 ~ 1=포화)
- `D` : 확산계수(매개충에 의한 공간 확산) — **PINN으로 역추정**
- `r` : 성장률(국소 증식), `u(1-u)`는 수용능력에 의한 포화

신경망 `N(x,y,t)→u` 가 ① 연도별 관측 감염밀도(데이터 손실)와 ② PDE 잔차(물리 손실)를
동시에 만족하도록 학습하며 미지 파라미터 `D, r`을 함께 추정합니다.

## 학습 / 검증 / 예측 구성
- **학습**: 2016–2022 (조사일자 1~12월 **완전 연도** 7개)
- **검증**: **2023** (1~4월 **부분관측** — 새로 확보한 실제 자료로 공간패턴 검증, 상관 r≈0.47)
- **예측**: 2024–2026 (물리법칙을 따른 외삽)
- **확산 예측**: ~2030 — 학습한 `D`로 **소진(burn-out) 고려 공간 SIR**을 수치 적분 (감염→제거 후 재감염 없음, 전선 이동형)

```bash

python -m pip install -r requirements.txt

# 2) Jupyter 커널 등록 (선택)
python -m ipykernel install --user --name kpp --display-name "Python (KPP)"

# 3) 노트북 실행
jupyter notebook pine_wilt_KPP_PINN.ipynb

    --ExecutePreprocessor.timeout=1800 pine_wilt_KPP_PINN.ipynb
```

  경계 GeoJSON을 `assets/` 에 자동 다운로드합니다.

## 주요 결과 (해석)

- 피해는 **남동부(경북·경남·부산·울산)** 에 가장 집중되며 수도권·충청·제주에 부차 발생지 — 실제 분포와 부합.
- 학습연도 공간상관(관측 vs 재현) **0.70–0.87** 로 발생 구조를 잘 재현.
- 학습된 `D ≈ 15.5 km²/년`, `r ≈ 0.005 /년`(거의 0) → 총량이 **증가가 아닌 감소** 국면이라 성장률이 작게 추정됨.
  공간적으로는 기존 발생지에서 **지속·완만 확산되는 풍토병적 패턴**.
- 미래 예측(2024–2026)은 남동부 진원지 지속을 보이며 완만히 확산.
- **소진 고려 SIR 확산(~2030)**: 감염목이 고사·제거되면 그 자리는 감수성 기주가 사라져 재감염되지 않으므로, **활성 감염은 전선이 이동하며 핵심부가 소진**된다. 전국 평균 활성감염은 2027~28년 정점 후 감소(0.03→0.09→0.08), **누적 피해지역은 단조 증가**(~0.37). 장기 외삽 대신 단기(2030) 시사점으로 해석.
- 시도별로는 **경북 > 경남 > 울산 > 경기 > 제주** 순으로 피해 집중(제주는 증가 추세).
